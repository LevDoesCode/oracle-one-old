## Run java file
* Create a file
  ```java
  public class Test {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
  }
  ```
* Build the file to bytecode
  
  `javac myfile.java`
* Run the file (doesn't need .class extension
  
  `java myfile`
